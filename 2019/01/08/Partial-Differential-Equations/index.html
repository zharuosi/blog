Introduction

PDE can be classified as elliptic, parabolic or hyperbolic. In fluids dynamics, the governing equations contain first and second derivatives in the spatial coordinates and first derivatives only in time. The spatial derivatives often appear nonlinearly while the time derivatives appear linearly. For linear PDE of second-order in two independent variables xxx and yyy,

A∂2u∂x2+B∂2u∂x∂y+C∂2u∂y2+D∂u∂x+E∂u∂y+Fu+G=0A\frac{\partial^2 u}{\partial x^2} + B\frac{\partial^2 u}{\partial x \partial y} + C\frac{\partial^2 u}{\partial y^2} + D\frac{\partial u}{\partial x} + E\frac{\partial u}{\partial y} + Fu + G =0 A​∂x​2​​​​∂​2​​u​​+B​∂x∂y​​∂​2​​u​​+C​∂y​2​​​​∂​2​​u​​+D​∂x​​∂u​​+E​∂y​​∂u​​+Fu+G=0

A simple classification is as follows.

  • If B2−4AC<0B^2 - 4AC < 0B​2​​−4AC<0: elliptic PDE
  • If B2−4AC=0B^2 - 4AC = 0B​2​​−4AC=0: parabolic PDE
  • If B2−4AC>0B^2 - 4AC > 0B​2​​−4AC>0: hyperbolic PDE

In many cases, B=0B=0B=0. Under this condition,

  • If AC>0AC > 0AC>0: elliptic PDE
  • If AC=0AC = 0AC=0: parabolic PDE
  • If AC<0AC < 0AC<0: hyperbolic PDE

For a general case, a two-order PDE with NNN independent variables:

  • If all λ\lambdaλ are non-zero and all but one are the same sign, it leads to a hyperbolic PDE.
  • If any of λ\lambdaλ is zero, it leads to a parabolic PDE.
  • If all λ\lambdaλ are non-zero and of the same sign, it leads to an elliptic PDE.

Hyperbolic PDE

A simple example is the wave equation.

∂2u∂t2−c2∂2u∂x2=0\frac{\partial^2 u}{\partial t^2} - c^2\frac{\partial^2 u}{\partial x^2} = 0 ​∂t​2​​​​∂​2​​u​​−c​2​​​∂x​2​​​​∂​2​​u​​=0

The equation has the property that, if uuu and ∂u∂t\frac{\partial u}{\partial t}​∂t​​∂u​​ are arbitrarily specified initial data on the line t=0t = 0t=0 (with sufficient smoothness properties), then there exists a solution for all time ttt. The solutions of hyperbolic equations are “wave-like”. Disturbances of the initial or boundary conditions have a finite propagation speed. On the contrary, A perturbation of an elliptic or parabolic equation is felt at once by essentially all points in the field.
